# Interactive Personal Blog Platform

## Description
A client-side personal blog application built with vanilla JavaScript, featuring dynamic DOM manipulation, form validation, and localStorage persistence.

## How to Run
1. Open `index.html` in a modern web browser
2. Start creating, editing, and managing your blog posts

## Features
- Create new blog posts
- Edit existing posts in a modal
- Delete posts with confirmation
- Data persistence using localStorage
- Form validation with custom error messages
- Responsive design

## Technologies Used
- HTML5
- SCSS (compiled to CSS)
- Vanilla JavaScript
- localStorage API

## Development Process
This project is being developed incrementally with detailed commits tracking each feature implementation.
